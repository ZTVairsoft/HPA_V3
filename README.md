# Прошивка для ВВД контроллера gen3.0

Прошивка для плат 3 поколения
Программная часть является открытой и при в дальнейшем будет дописываться для внесения нового функционала
На платах имеются 3 нагрузки которые можно использовать на своё усмотрение.
1 нагрузка обозначена на плате как СОЛ1 - предназначена для подключения соленоида вашей системы, является самым высокотоковым элементом способным открыть высокотоковые соленоиды
2 нагрузка обозначена на плате как СОЛ2/НАГ1 - в случае подключения двухсоленоидной системы предназначена для подключения соленоида отвечающего за выдвижение нозла или при использовании односоленоидной системы может использоваться как источник для питания периферии на подобии трассерного хопа или к примеру бубна пулемета
3 нагрузка обозначена на плате как НАГ2 - Так же как и 2 нагрузка предназначена для подключения дополнительной периферии.
Предусмотрено подключение 3-х микропереключателей отвечающих за:
Спусковой крючек - представлен в нескольких вариациях возможных к применению микриков идущих в комплекте
Переключатель режимов огня - представлен не во всех платах, отсутствует в основном там где он не нужен, к примеру, на пулемете
Электронный предохранитель - является вариативным элементом и по умолчанию отключен, актуален для спидсофтеров у которых ход спуска слишком короткий что бы физических предохранитель мог остановить нажатие.
